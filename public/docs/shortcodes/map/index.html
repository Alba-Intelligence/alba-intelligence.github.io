Map

A map is basically an image with a marker on top of it. The marker can be found in the ‘/static/img/’ directory. Make sure to make your map image large enough, as it does not scale. The minimum size is a 900x400 pixel image. You can see the map shortcode in action on the Elements page. To show the map in your content, write the following shortcode:

{{< map "/uploads/map.png" "https://link.to.map.location" >}}
